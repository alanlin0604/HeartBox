# HeartBox PWA 安裝指南

HeartBox 是一款 Progressive Web App (PWA)，可以安裝到手機或電腦桌面，像原生 App 一樣使用。

---

## iOS（Safari）

1. 用 **Safari** 開啟 HeartBox 網址
2. 點底部的 **分享按鈕**（方框 + 箭頭圖示）
3. 往下滑找到 **「加入主畫面」**
4. 點「新增」，桌面就會出現 HeartBox 圖示
5. 打開後會全螢幕運行，體驗與原生 App 相同

> **注意：** iOS 僅支援 Safari 安裝 PWA，Chrome / Firefox 無法使用此功能。

---

## Android（Chrome）

1. 用 **Chrome** 開啟 HeartBox 網址
2. 瀏覽器通常會自動彈出 **「新增至主畫面」** 提示
3. 如果沒有彈出，點右上角 **⋮ → 安裝應用程式**
4. 確認安裝，桌面就會出現 HeartBox 圖示

---

## 電腦（Chrome / Edge）

1. 開啟 HeartBox 網址
2. 網址列右側會出現 **安裝圖示**（⊕ 或下載圖示）
3. 點擊後確認安裝
4. 會在桌面 / 開始選單建立捷徑，開啟後以獨立視窗運行

---

## 安裝後的好處

| 功能 | 說明 |
|------|------|
| 離線瀏覽 | 已快取的頁面在無網路時仍可檢視 |
| 推播通知 | 接收預約提醒、諮商師回覆等通知 |
| 全螢幕體驗 | 隱藏瀏覽器工具列，如同原生 App |
| 桌面捷徑 | 從主畫面 / 桌面一鍵啟動 |
| 自動更新 | 每次開啟時自動載入最新版本 |

---

## 技術細節

HeartBox PWA 由以下元件組成：

- **`manifest.json`** — 定義 App 名稱、圖示、啟動 URL、主題色等
- **`sw.js`**（Service Worker）— 負責快取管理、離線支援、推播通知
  - 導覽請求：Network-first 策略（優先取得最新內容）
  - 靜態資源：Cache-first 策略（優先使用快取加速載入）
  - API / WebSocket / 媒體請求不經過快取
- **自動更新機制** — Service Worker 版本更新時自動啟用新版並重新載入頁面

---

## 解除安裝

- **iOS：** 長按主畫面圖示 → 移除 App
- **Android：** 長按圖示 → 解除安裝（或至設定 → 應用程式中移除）
- **電腦：** 開啟 App → 點標題列 ⋮ → 解除安裝
