認知行為療法（CBT）技巧指南

一、什麼是認知行為療法？

認知行為療法（Cognitive Behavioral Therapy, CBT）是一種以實證為基礎的心理治療方法，核心理念是：我們的想法（認知）、感受（情緒）和行為三者相互影響。透過辨識並修正不合理的思維模式，可以改善情緒和行為反應。

CBT 被世界衛生組織推薦為治療憂鬱症、焦慮症、恐慌症等常見心理困擾的第一線治療方法。

二、常見的認知扭曲

認知扭曲是指不合理或偏誤的思維模式，常見類型包括：

1. 全有或全無思維：事情只有完美或失敗兩種結果，沒有中間地帶。
   例如：「這次報告沒有拿到滿分，我就是個失敗者。」

2. 過度概括化：從一次事件就推論出普遍性的結論。
   例如：「他今天沒回我訊息，代表沒有人在乎我。」

3. 心理過濾：只注意負面細節，忽略正面經驗。
   例如：一整天收到十個讚美，但只記得那一個批評。

4. 否定正面經驗：把正面事件解釋為不算數或沒意義。
   例如：「他只是客氣才稱讚我，不是真的覺得我做得好。」

5. 讀心術：不經驗證就假設別人在想什麼。
   例如：「她一定覺得我很笨。」

6. 預言式思考：預測最壞的結果一定會發生。
   例如：「明天面試一定會搞砸。」

7. 災難化：把小問題放大成災難。
   例如：「遲到五分鐘，老闆一定會開除我。」

8. 應該式思維：用「應該」、「必須」來要求自己或他人。
   例如：「我應該要隨時保持開心，不應該感到難過。」

三、思維記錄法（Thought Record）

思維記錄法是 CBT 最核心的技巧之一，幫助你系統性地檢視和調整想法。

步驟：
1. 描述情境：什麼時候、什麼地方、發生了什麼事？
2. 辨識情緒：你當時感受到什麼情緒？強度如何（0-100）？
3. 捕捉自動化思維：當時腦海中浮現什麼想法？
4. 找出認知扭曲：這個想法屬於哪一種認知扭曲？
5. 產生替代思維：有沒有更平衡、更合理的想法？
6. 重新評估情緒：用新的想法重新感受，情緒強度是否有變化？

實際範例：
- 情境：同事在會議上指出我報告中的錯誤
- 情緒：丟臉（80）、沮喪（70）
- 自動化思維：「我太差勁了，大家一定覺得我不專業」
- 認知扭曲：過度概括化、讀心術
- 替代思維：「報告有錯是正常的，同事指出問題是在幫助我改進，這不代表我整個人不專業」
- 重新評估：丟臉（30）、沮喪（20）

四、行為實驗

行為實驗是用來驗證你的想法是否符合事實的方法。

步驟：
1. 找出你想驗證的想法：例如「如果我拒絕別人的請求，他們會討厭我」
2. 設計實驗：下次有不合理的請求時，試著禮貌地拒絕
3. 預測結果：寫下你預期會發生什麼
4. 執行實驗：實際去做
5. 記錄結果：實際發生了什麼？
6. 反思學習：結果和預期一樣嗎？學到了什麼？

五、活動安排（Behavioral Activation）

當情緒低落時，人往往會退縮、減少活動，形成惡性循環。活動安排的目的是打破這個循環。

具體做法：
1. 記錄每天的活動和對應的心情分數（0-10）
2. 找出哪些活動能提升心情
3. 有意識地安排更多讓自己感覺好的活動
4. 從小目標開始，不需要一次做很多
5. 即使不想做也試著開始，因為行動往往會帶動情緒改善

建議的正向活動清單：
- 散步或輕度運動
- 和朋友聊天或見面
- 聽喜歡的音樂
- 閱讀或看電影
- 做一件小小的善事
- 整理房間的一個角落
- 嘗試一道新料理
- 寫下三件感恩的事

六、漸進式暴露

對於恐懼和焦慮，漸進式暴露是有效的方法：

1. 列出你害怕的情境，從最不害怕到最害怕排序
2. 從最容易的開始練習
3. 在每個層級待到焦慮下降至少一半
4. 再進入下一個層級
5. 重複練習，直到不再感到過度焦慮

七、日常練習建議

1. 每天花 5-10 分鐘寫思維記錄
2. 注意自己的自動化思維，嘗試辨識認知扭曲
3. 練習用「朋友的角度」看待自己的問題
4. 設定小而具體的行動目標
5. 記錄成功的經驗，建立正面的自我認知

提醒：CBT 技巧需要持續練習才能見效，就像學習任何新技能一樣。如果情緒困擾嚴重，建議尋求專業心理師的協助。
